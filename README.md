## 🇷🇺 Русская версия

## Обзор проекта

**Единая Бонусная Ссиетма** - это проект мобильного приложения, посвящённый управлению системой лояльности для кафе, кофеен, ресторанов и других заведений общественного питания.

Смысл проекта простой: дать клиенту удобный способ **получать, отслеживать и использовать бонусы** в одном месте, а бизнесу — более прозрачную и организованную систему поощрения клиентов.

Проект разрабатывается как **учебная курсовая работа**, но при этом строится вокруг практической идеи и в будущем может перерасти в более полноценный продукт.

---

## Зачем нужен проект

Многие бонусные системы либо слишком ограничены, либо неудобны для клиентов, либо запутаны в использовании.  
Цель этого проекта — решить проблему через единое мобильное приложение, где:

- клиенту удобно работать со своими бонусами;
- начисление и списание бонусов становится прозрачным;
- механика лояльности становится понятнее;
- бизнес получает инструмент для удержания клиентов.

---

## Основные цели

- Создать понятный и удобный мобильный интерфейс

## 🇺🇸 English

## Overview

**Unified Bonus System** is a mobile application project focused on loyalty program management for cafes, coffee shops, restaurants, and similar businesses.

The idea of the project is simple: give customers a clear and convenient way to **earn, track, and redeem bonuses** in one place, while giving businesses a more transparent and organized reward system.

This project is being developed as an **educational coursework project**, but it is designed with a practical idea behind it and can grow into a more complete product in the future.

---

## Why this project exists

Many loyalty systems are either too limited, confusing for users, or inconvenient to manage.  
This project aims to solve that by building a single mobile solution where:

- customers can easily interact with their bonus balance;
- bonus accrual and redemption are transparent;
- loyalty mechanics are easier to understand;
- businesses can improve customer retention.

---

## Main goals

- Build a clear and user-friendly mobile interface
- Implement the core bonus system logic
- Create a solid foundation for future expansion
- Practice real application structure and development workflow
- Combine educational development with a practical business idea
